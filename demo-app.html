ReadShift Installation Guide

ReadShift runs powerful AI models directly on your computer for maximum privacy and offline use. Please follow these steps to set up your local environment.

1. Install Ollama

Ollama is the tool that runs the AI models. Download and install it from the official website: https://ollama.com/

2. Download the AI Model

Once Ollama is installed, open your computer's terminal (Terminal on macOS/Linux, PowerShell or Command Prompt on Windows) and run this command:

ollama run llava

We recommend the 'llava' model because it can understand both images and text, which is needed for the camera feature.

3. Start Using ReadShift

With Ollama running, you are ready to go! Launch the app and start learning.
